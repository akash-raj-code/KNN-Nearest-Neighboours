# 🤖 ML Series 1 — Machine Learning Notebooks

A series of in-depth, hands-on Jupyter notebooks covering core machine learning algorithms with intuition, visualizations, and practical implementations.

> Based on [Krish Naik's ML Playlist](https://www.youtube.com/playlist?list=PLZoTAELRMXVPBTrWtJkn3wWQxZkmTXGwe)

## 📚 Notebooks

| # | Notebook | Topic | Key Concepts |

## 🧠 Topics Covered in KNN Notebook

- ✅ What is KNN and how it works (intuition)
- ✅ Distance Metrics: Euclidean, Manhattan, Minkowski
- ✅ KNN Classifier — built **from scratch** in Python
- ✅ KNN Regressor — built **from scratch** in Python
- ✅ Finding Optimal K using Cross-Validation (Elbow Method)
- ✅ Decision Boundary Visualization
- ✅ Weighted KNN (`uniform` vs `distance`)
- ✅ Feature Scaling — why it's **mandatory** for KNN
- ✅ Curse of Dimensionality Demo
- ✅ Hands-on: **Iris Dataset** (Classification)
- ✅ Hands-on: **Diabetes Dataset** (Regression)

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/<your-username>/ML-Series-1.git
   cd ML-Series-1
   ```

2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Launch Jupyter Notebook**
   ```bash
   jupyter notebook
   ```

4. Open the notebook and run cells top-to-bottom ▶️
